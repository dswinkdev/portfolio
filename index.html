Original Code (C++)

ABC University was developed as a C++ project, that allowed users to input course data and retrieve information. The search function is based on a Binary Search Tree, providing efficient data retrieval.

Enhanced Code (Java)

With improved performance and functionality, the enhanced prototype represents a polished and scalable event tracking experience. One of the most significant upgrades was the implementation of an AVL tree, a self-balancing binary search tree that ensures optimal time complexity for insertions, deletions, and lookups. This data structure maintains strict balance by automatically adjusting its height after each operation, which guarantees O(log n) performance. As a result, the application can now handle larger volumes of course data more efficiently, providing academic advisors with faster and more reliable access to course details and prerequisites.
