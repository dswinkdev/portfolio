Course Navigator Code Review

Original Artifact

The initial version of Course Navigator was a simple text-based C++ app that printed course data in order. It lacked error handling and assumed data files were perfectly formatted with no edge-case considerations.

Original Console App

Enhancement

The project was restructured to support file validation, modular pseudocode logic, prerequisite verification, and the use of three data structures: vectors, hash tables, and binary search trees. These upgrades significantly improved runtime and maintainability.

Improved Program Structure

Future Enhancements

Plans include building a web-based interface for live course lookup, integrating backend validation using a real-time database, and allowing export features for advising reports. This would further streamline advisor workflows and enhance student planning tools.
